Login Use Case

Use Case Text

Successful Login

User System
1. System displays login menu
2. User enters username and password, then selects 'Login' [ALT-1]
3. System validates username and password [ALT-2]
4. System displays menu
[ALT-1] User selects 'Create New User'
Continue with Create New User		 [ALT-2] Invalid credentials
Display 'Login failed' message

Create New User

User System
1. User selects 'Create New User'
2. System displays 'Create User' window
3. User enters username and password [ALT-1]
4. System verifies username and password [ALT-2]
5. System creates User using username and password
6. System opens menu
[ALT-1] User selects 'Cancel'
Return to Login Menu		 [ALT-2] Invalid username and/or password
Display error on 'Create User' window
